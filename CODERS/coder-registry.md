tags: #coders #registry #agents

---

## Purpose

This registry tracks coding agent lanes and what each one is expected to do well.

| Coder Lane | File | Best Use | Main Limitation |
|---|---|---|---|
| Codex Local | `codex-local.md` | Local repo edits, path verification, desktop-visible systems | Cannot see anything not mounted or permissioned in the local session |
| Codex Online | `codex-online.md` | Cloud/worktree/GitHub-connected code changes | Cannot assume access to David's local drives, NAS, vault, or services |

## Rule

Add a new lane only when it changes how coding work should be handled. Do not make personality files for their own sake.
